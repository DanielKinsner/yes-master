# Reference Docs

These files are optional context from the Codex/current-repo path.

They are not startup reading for Claude. Use them only if the user explicitly asks for Codex context or if a narrow historical detail is needed. They should not steer this build toward the current implementation.

Files:

- `codex-implementation-plan.md`: the current Codex repo execution plan.
- `codex-research-implementation-notes.md`: notes on research already applied in the current repo.

Prefer `docs/PRODUCT.md`, `CLAUDE.md`, `docs/CLAUDE_BUILD_BRIEF.md`, `docs/CLAUDE_WORK_LOOP.md`, and the research docs when making decisions for this clean build.
